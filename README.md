Chat Corporativo Médico – Relatório e Instruções de
Uso
Descrição do Projeto
Este projeto consiste em uma aplicação web de chat corporativo para ambiente médico,
desenvolvida utilizando Node.js, Express e WebSocket. O sistema permite comunicação em
tempo real entre profissionais da saúde como médicos, enfermeiros, recepcionistas e equipe
administrativa dentro de uma clínica ou hospital.
Objetivo do Sistema
O objetivo do sistema é facilitar a comunicação interna da clínica, permitindo troca de
informações rápidas entre os profissionais, como avisos de pacientes, comunicação entre
médicos e enfermeiros e organização do atendimento.
Tecnologias Utilizadas
Node.js, Express, WebSocket, HTML, CSS e JavaScript.
Como Executar o Sistema
1. Instalar o Node.js no computador.
2. Abrir o terminal dentro da pasta do projeto.
3. Instalar as dependências executando:
npm install express ws
4. Iniciar o servidor com o comando:
node server.js
5. Após iniciar o servidor, abrir o navegador e acessar:
http://localhost:3000/login.html
Como Criar um Usuário
1. Na tela inicial clique em 'Criar conta'.
2. Digite um nome de usuário.
3. Digite uma senha.
4. Clique em 'Criar Conta'.
5. Após o cadastro você poderá voltar para a tela de login.
Como Fazer Login
1. Acesse a página de login.
2. Digite o usuário cadastrado.
3. Digite a senha.
4. Clique no botão 'Entrar'.
5. O sistema abrirá a página inicial do sistema.
Como Acessar o Chat
1. Após fazer login, clique no botão 'Entrar no Chat'.
2. A página do chat corporativo será aberta.
3. Digite a mensagem no campo de texto.
4. Pressione 'Enviar' ou a tecla Enter para enviar a mensagem.
Testando com Vários Usuários
Para testar múltiplos usuários conectados ao mesmo tempo:
• Abra o sistema em navegadores diferentes (Chrome, Edge, Firefox), ou
• Use uma janela anônima do navegador.
Cada usuário poderá fazer login com uma conta diferente e conversar no chat.
Benefícios do Sistema
• Comunicação rápida entre setores da clínica
• Organização das informações
• Redução de falhas de comunicação
• Integração da equipe médica
